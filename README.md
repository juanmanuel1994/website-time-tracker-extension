Time Tracker - Chrome Extension
================================

A Chrome extension that automatically tracks how much time you spend on each website throughout the day.

HOW IT WORKS
------------
The extension runs silently in the background, recording time per domain as you browse. It pauses automatically when you are idle (no activity for 60 seconds) and resumes when you return. Time is stored locally and resets daily for the "Today" view.

FEATURES
--------
- Tracks time per website automatically, no setup needed
- Shows current active site and live time counter
- Top 8 most visited sites ranked by time with progress bars
- Switch between Today view and All Time view
- Total browsing time summary at the bottom
- Reset button to clear all recorded data
- Ignores browser sleep and long pauses to keep stats accurate

USAGE
-----
1. Click the extension icon to open the dashboard
2. View your current site and time spent on it in real time
3. Scroll through your top sites ranked by time
4. Toggle between Today and All Time using the buttons at the top
5. Click Reset to clear all statistics and start fresh
